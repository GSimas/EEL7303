
placement  [options] <input design> [<output design>] 

	Allegro auto-place program

  options:
    -a           improve while (percent < ACCEPT)
    -w           weight the complete graph edges
    -p           print the connection matrix
    
  <input design> - name of input design
  <output design> - optional, output file. If not provided
		will overwrite input design

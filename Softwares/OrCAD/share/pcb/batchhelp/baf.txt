
Command line arguments for baf
	Outputs third party netlists from Allegro

   baf [-s] [-c] <compare.design_name> <source.design_name>

	compare.design_name and source.design_name can be same name otherwise 
	compare.design_name is an older version of design

	Where design is a .brd, .mcm or .sip

   Arguments:
      -s    Include spare T-F functions 
      -c    Output Electrical data (cmdbview.dat)


Command line arguments for ipc2581_in. 
        Utility to import IPC2581 data to Allegro.

ipc2581_in <ipc2581 data file> [-o <output allegro database>]
		[-i <input allegro database>  


    OPTIONAL ARGUMENTS      

        [-o <output board file>]
              Output board file name. 
              Default: <ipc2581_data_name>.brd

        [-i <input board file>]
              Input board file name. 
              Default: uses output allegro database (overwrites file)       
       
    REQUIRED ARGUMENTS

        <ipc2581 data file>
              Name of the IPC2581 Data File.

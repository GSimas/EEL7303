Adding Meta Tags to HTML Documents


You need to add the following meta tags to the <HEAD> section of the documents you want to list in Cadence Help:

<META NAME="DocType" CONTENT="Manual">

Required: Identifies that this is a manual (and not a Cadence Known Problems and Solutions or Product Note document)

<META NAME="DocTitle" CONTENT="}}My Manual Name{{">

Required: Specifies the name of the document as listed under the manual heading in the Cadence Help window.

<META NAME="FileTitle" CONTENT=" Table of Contents ">

Optional: Specifies the name of this file.

<META NAME="Product" CONTENT=" Product Name ">

Required: Specifies the name of the product that will appear for this book when you expand products in the Cadence Help window.

<META NAME="FileType" CONTENT="TOC">

Optional: Specifies the file as a Table of Contents or Chapter.

<META NAME="ProductFamily" CONTENT=" Cadence Product Family Name ">

Optional: Specifies the Cadence Product Family under which this document will appear when you expand platform in the documentation browser.

<META NAME="Group" CONTENT=" Group Name ">

Optional: Specifies the group under which the book will appear (within a product) in Cadence Help window. In case the group is not specified, the books will appear under their respective products.

<META NAME="Order" CONTENT=" 1 ">

Optional: Specifies the order in which the book will appear (within a group or Product) in Cadence Help window. In case the order is not specified, the books will appear in the alphabetical order.

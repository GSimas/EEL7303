Cadence Help Interface Overview

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

To perform a search, type a term in the search box and press , or use Search and View Options to refine your search.

The Advanced Search options can help you find precisely what you are looking for.

You can explore the documentation installed on your machine from the Documentation Browser area.

The Topic Viewer area displays the search results as well as individual topics from the help documents.

The various features of Topic Toolbar can be used to optimize your viewing experience.

You can use tabs to open and view multiple topics simultaneously.

The library feature enables you to add or delete libraries, hide or show libraries, or change the order of their appearance in the documentation browser.

You can use the menu options to perform tasks such as importing and exporting documents, modifying the settings, viewing search history, and organizing favorites.


For support, see Cadence Online Support service.

Layout to PCB Editor dialog box

To open this dialog

In the Project manager, choose Layout to PCB Editor from the Accessories - Layout to PCB Editor translator menu.

Use this control...

To do this...

Input Layout file (.max)

Specify the layout design (.MAX) file.

Output directory

Specify where you want to store the output files, such as the translated brd file and the log files.

Overwrite brd and associated files files

Select to overwrite existing files. If you do not select this option, a message will be displayed if files exist.

Update dsn with brd

Select this option to synchronize the design file with the board file that is created after the translation process.

View Log

Select to view the log files. If you select the Update dsn with brd option, the synchronization log will display. If this option is not selected, the translation log will display. You can also view the synchronization log in the Session window.

Translate

Click to start the translation process.

Library Management dialog box

To open this dialog

Click Launch Library Management in the Library Setup (SI Analysis) dialog box (SI Analysis - SI Library Setup)..

Use this control...

To do this...

Working Library

Select to specify the library as the working library.

Ignore Library

Select to ignore the library.

Launch Model Integrity

Click to launch Model Integrity.

Library Setup (SI Analysis) dialog box

To open this dialog

In the Project manager, choose SI Analysis - SI Library Setup.

Use this control...

To do this...

 

Click to add a new library.

 

Click to remove the selected library from the list.

 

Click to open the Library Management dialog box.

 

Click to move up the selected library in the list.

 

Click to move down the selected library in the list.

Working

Displays the name of the working directory.

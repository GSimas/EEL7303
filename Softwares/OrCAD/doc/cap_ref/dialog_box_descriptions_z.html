Zoom Scale dialog box

To open this dialog

In the schematic page editor, choose Zoom - Scale from the View menu.

Use this control...

To do this...

X%

Choose a pre-defined zoom scale ranging from 25% to 400%.

Custom X%

Specify a custom zoom scale.

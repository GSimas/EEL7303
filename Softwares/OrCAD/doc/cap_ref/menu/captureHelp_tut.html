OrCAD Capture Tutorial

Self-paced interactive tutorial that you can use to quickly get started with OrCAD Capture. You can practice what you've learned by going through the tutorial's specially designed exercises that interact directly with Capture. You can access this tutorial from Help - Learning OrCAD Capture from the OrCAD Capture window.
The lessons and exercises run from the browser window and require the one-time installation of the Adobe Flash Player 10.0 or later version.
For more information on the latest free Flash Player downloads, see the Adobe Web Players page. Ensure that the default browser is specified in Cadence Help.
Note: In case you have problems running the tutorial in your default browser, please check that the path to the tutorial (the installation path) is a mapped drive and not a UNC path.

OrCAD Flow Tutorial

Describes the design cycle for an electronic design, starting with capturing the electronic circuit in OrCAD Capture, simulating the design with PSpice, through the PCB layout stages in OrCAD Layout / OrCAD PCB Editor, and SPECCTRA, and finishing with the processing of the manufacturing output and maintaining the design through ECO cycles.

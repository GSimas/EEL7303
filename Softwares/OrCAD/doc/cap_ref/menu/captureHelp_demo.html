This page lists multimedia demonstrations (demos) for important features in OrCAD Capture.

These demos run as stand-alone files from the browser window and require the one-time installation of a Flash plugin. To view the Cadence Silicon-Package Board demonstrations, you need to install a Flash player. For more information on the latest free Flash Player downloads, see the Adobe Web Players page.

Creating differential Pairs between multiple pairs of flat nets in a design simultaneously

Creating a New Part from Spreadsheet

Using the New Part Creation Spreadsheet to create and add new parts to a library.

Creating a Zip Archive

Creating a zip archive of your OrCAD Capture project and related files.

Signal Properties Flow from Capture to Allegro PCB Editor

Assigning signal properties to nets.

Creating a Bill of Materials

Creating a bill of materials for a OrCAD Capture design.

Associating a PSpice Model to a Capture Symbol

Associating a PSpice model to a Capture Symbol using the Associate PSpice Model command.

Customizing part references in a design

Customizing the way OrCAD Capture assigns part references in a design.

Advanced Analysis Workflow

Advanced Analysis is an add-on program for PSpice A/D and AMS Simulator1. Use these four Advanced Analysis tools to improve circuit performance, reliability, and yield:

 Sensitivity identifies which components have parameters critical to the measurement goals of your circuit design. For more information see The Sensitivity Tool .
The four Optimizer engines optimize the parameters of key circuit components to meet your performance goals. For more information see The Optimizer Tool .
Smoke warns of component stress due to power dissipation, increase in junction temperature, secondary breakdowns, or violations of voltage / current limits. For more information see The Smoke Analysis Tool .
Monte Carlo estimates statistical circuit behavior and yield. For more information see The Monte Carlo Tool .

Simulation Tab

The Simulation tab of the Profile Settings dialog box allows you to control the amount of data stored in the simulation results data file. You can specify the number of runs for which data is to be stored for Sensitivity, Optimizer, and Monte Carlo. You can specify the following two options:

 Save All Runs: Data for all runs is stored in the data file.
Save None: Data is saved only for the last run.

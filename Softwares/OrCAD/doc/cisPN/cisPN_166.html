Configuring Menus and Toolbars

You can now customize the menus and toolbars in OrCAD Capture CIS. As a result, any TCL methods that you want to be able to run from the menus is possible. You can also specify your own icons for the menus or toolbar items. The resource files for menus and toolbars including the icons are located at:

<Cadence_installation>\share\orResources

You can add menus through XML files where you specify:

 the menu label
name of TCL method to be called on menu click
location of the menu in existing menu items
TCL method to enable and grey out the menu item
an optional icon

You can also add dynamic menu items using TCL code.

Note: The pop-up menus cannot be customized.

Full RefDes support

In 16.6 OrCAD Capture and OrCAD Capture CIS handle references and designators in the same manner. As a result, both the tool handle RefDes values similarly.

You will now see similar references in all reports generated from the two tools. In addition, CIS will not delete leading zeros if they are part of a RefDes.

Multivalue Support

You can now set any field as multivalued. To set a field as multivalued, launch the CIS Configuration File (Options - CIS Configuration) and click Setup to display the Database dialog box.

You can select a delimiter value from the Delimiter for Multi-Values list in the Administrative Preferences tab. You can specify values separated by the specified delimiter in a multivalued field.

Performance Enhancements

Performance has been enhanced in the invocation of Part Manager and CIS BOM and SWL Query optimizations.

Linking Part Occurrence

You can now link part occurrences at group and subgroup level.

Linking External Design Parts

You can now link parts of external design at the group or subgroup level. The external parts are linked at the occurrence level within the currently active design.

You can also link parts at instance level by choosing the Link Instance option from the pop-up menu if there is only one hierarchical block of the external design schematic in the active design.

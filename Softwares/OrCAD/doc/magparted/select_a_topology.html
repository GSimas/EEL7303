Magnetic Parts Editor Help


Selecting a component

This is the first step for designing a transformer for Switch Mode Power Supplies (SMPS). In this step, you select the type of magnetic component to be designed using Magnetic Parts Editor. The topologies supported by Magnetic Parts Editor are:

 
Power transformers
 
Forward Converter (Single and Double Switch)
 
Flyback Converter (Discontinuous conduction mode)
 
DC inductors.

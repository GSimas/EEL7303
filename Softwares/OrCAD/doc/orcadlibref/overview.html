OrCAD Supplied Libraries




Overview

Use this help file to locate OrCAD supplied library parts. Each part and library supplied by OrCAD appears as an index keyword in this help file. You can locate a part within a library by using the part name as a keyword. You can also see what parts are found in a library by using the library name as a keyword. Some parts are vendor specific. In these cases, the vendor name is also listed as a keyword.

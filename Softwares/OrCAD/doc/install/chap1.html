About This Guide

The Cadence Installation Guide provides the basic information and procedures required to install Cadence® products. Though Appendix A, "Overview of Software Configuration" provides a brief summary of software configuration, refer to your product documentation for detailed configuration information.

The most effective way to use this guide is for you to determine your level of experience with system administration and Cadence product installation. Users who have little experience with system administration and have not installed Cadence products should read the entire guide. Users who have experience with system administration might want to use the Cadence Installation Quick Reference that is provided as a CD-ROM insert.




Additional Information



Cadence Installation Quick Reference

The Cadence Installation Quick Reference is a printed document included in the CD-ROM case. It briefly outlines how to install Cadence products. It contains no details on licensing, configuration, or testing.



InstallScape User Guide

The InstallScape User Guide containing information on how to configure and use InstallScape is available at the top level of your CD. It is strongly recommended that you go through this document and familiarise yourself with InstallScape before starting installation of Cadence software.



Release-Specific Information

A README file containing release- (or product-) specific information might exist for your product. If this file exists, you will find it in the top level of your CD.

Capture Command Line Arguments & Switches

You can invoke OrCAD Capture directly from the command line with specific arguments and switches.

Syntax

capture [-i <capture.ini location>] [dsn name] [tcl script name]

 

Switch / Argument

Description

-i <capture.ini location>

The location of the capture.ini to be used with the current instance of Capture.

dsn name

Name and location of the dsn to be opened in the current instance of Capture.

tcl script name

Name and location of the tcl file to be executed at startup.

Note:  

 
All the arguments are optional.
 
The dsn name and captcl name arguments cannot be used in conjunction with each other.

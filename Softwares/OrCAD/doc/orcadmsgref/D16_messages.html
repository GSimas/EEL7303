[D16nnnn] Messages

The [D16nnnn] messages are DECOMP.EXE errors and warnings for 16-bit libraries. Each of these messages may be caused by more than one type of problem, but end up with the same result. In general, you should check to make sure that all files you work with are not read-only or corrupted. For parts, pins, nets, and properties, check that you are using valid character names. If Capture is unable to allocate memory for a task, check to see if you have sufficient system resources for Capture. You may need to close down applications that you are not using.

PSpice Modeling Application: Non Ideal RF Inductors

This device would model a Non Ideal RF Inductor which exhibits resonance. You need to define Inductance, internal DC Resistance, Q Factor, Self Resonant frequency and Tolerance. Each of the parameters can be obtained directly from the inductor datasheet. This model includes an additional resistance in parallel, which on selection is included in parallel to capacitor of basic inductor model. This model doesnot include core model.

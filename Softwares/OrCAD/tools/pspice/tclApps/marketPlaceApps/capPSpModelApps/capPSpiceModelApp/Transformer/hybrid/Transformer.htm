Two Winding Transformers: This device would model basic two winding transformer with ideal coupling. You need to define primary winding inductance, primary winding resistance, turn ratio and mutual coupling. Turn ratio is defined as N2/N1; thus for step up transformer with step up ratio of 10, turn ratio should be 10 and for step down ration of 10 it should be 0.1. Coupling represents the coefficient of mutual coupling, which must be between 0 and 1.0.This coefficient is defined by the equation

"coupling value" = Mij/SQRT(Li·Lj)
where,

Lij = a coupled-pair of inductors
Mij = the mutual inductance between Li and Lj

Multi Winding Transformers: This device would model a transformer where the number of secondary windings can be specified by user and this would be based on ideal coupling. You need to define primary winding inductance, primary winding resistance, and number of secondary windings and turn ratio for each secondary. Turn ratio is defined as N2/N1; thus for step up transformer with step up ratio of 10, turn ratio should be 10 and for step down ratio of 10 it should be 0.1. You also need to define mutual coupling between primary and secondary, which must be between 0 and 1.0.

Custom Tap Transformers: This device would model a custom tap transformer, where user can select the winding in which he needs to put tap and tap%. This transformer would be based on ideal coupling. You need to define primary winding inductance, primary winding resistance, winding in which you need to apply tap and turn ratio. Turn ratio is defined as N2/N1; thus for step up transformer with step up ratio of 10, turn ratio should be 10. Here turn ratio is applied on complete secondary winding. Secondary winding resistance is divided as per tap% and put at end of corresponding secondary winding. You also need to define mutual coupling between primary and secondary, which must be between 0 and 1.0.

Center Tap Transformers: This device would model a center tap transformer, where user can select the winding in which he needs to put tap. This transformer would be based on ideal coupling. You need to define primary winding inductance, primary winding resistance, winding, where which you need to apply tap and turn ratio. Turn ratio is defined as N2/N1; thus for step up transformer with step up ratio of 10, turn ratio should be 10. Here turn ratio is applied on complete secondary winding. Secondary winding resistance is equally divided at each end of secondary winding. You also need to define mutual coupling between primary and secondary, which must be between 0 and 1.0.

FlyBack Transformers: This device would model a single ended flyback converter transformer with linear core. You need to define primary winding inductance, primary & secondary winding resistance, leakage inductance and turn ratio. Leakage inductance value should be referred to secondary side. Turn ratio is defined as N2/N1; thus for step up transformer with step up ratio of 10, turn ratio should be 10.

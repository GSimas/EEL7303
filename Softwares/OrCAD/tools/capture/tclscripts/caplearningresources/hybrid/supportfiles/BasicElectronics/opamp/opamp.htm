Operational Amplifiers

This section discusses various basic circuits using operational amplifier (opamp) and explains how to simulate these in OrCAD® Capture-PSpice®.

An operational amplifier is a linear analog device with differential input and an amplified output. It usually has three terminals; two high-impedance inputs and one output. One of the input is termed inverting input and shown with a negative sign (-) and the other input is termed non-inverting input and shown with a positive or plus sign (+). The output is characterized by an amplification factor called gain (A) that denotes the amount by which the input is amplified.

Figure 1. Circuit notation

A circuit symbol for an opamp is shown in figure (1), where:

  • IN+ : non-inverting input
  • IN- : inverting input
  • OUT : output
  • V+ : positive power supply
  • V- : negative power supply

Voltage Follower

Voltage follower, also known as unity gain buffer, is an amplifier with a closed loop gain of unity. This configuration is used as a buffer amplifier to eliminate loading effects such as to connect a device with a high source impedance to a device with a low input impedance. The output voltage (Vout) of a voltage follower is equal to the input voltage (Vin) and the differential input impedance (Zin) is infinity as given in the following equations. Active Peak Detector

The active peak detector circuit detects the peak levels of the input signals and is created by adding diodes and capacitors to an operational amplifier. During the positive half-cycle of Vin, the output of the opamp drives the diode D1 ON charging capacitor C1 to the positive peak value VP of the input voltage Vin. In other words, when D1 is forward-biased, the opamp acts as a voltage follower. During the negative half-cycle of  Vin, the diode D1 is reverse-biased, and the voltage across C1 is retained. Since the input bias current is negligible, C1 discharges through R5 . For proper operation of the circuit, the charging time constant (CRd) and the discharge time constant (CRL)
must satisfy the following conditions:
CRd << T/10 and CRL =10T

where Rd is the forward resistance of the diode, RL is the load resistor and T is the time period of the input waveform.

Figure 22. Active peak detector circuit

It can be seen through the waveform, the circuit is able to detect the active peak whenever peak is present in the input signal.



Figure 23. Resultant waveform

Precision Rectifier

In a passive rectifier circuit, there is a voltage drop VF across the forward biased diode. In the active version shown in the figure, the diode is connected in the negative feedback loop to remove the voltage drop. The opamp compares the output voltage across the load with the input voltage and increases its own output voltage with the value of VF. As a result, the voltage drop VF is compensated and the circuit behaves very nearly as an ideal (super) diode with VF = 0V, this circuit configuration is known as precision rectifier.

Figure 24. Precision rectifier circuit

As can be seen voltage at the output is equal to input voltage, 5V, and there is no voltage drop.

Zero Cross Detector

A zero cross detector is a comparator with the reference level set at zero. It is used to detect the zero crossings of AC signals. The circuit consists of a comparator with an input voltage at the non-inverting input.
When the input voltage is positive, the output voltage is a positive value; when the input voltage is negative, the output voltage is a negative value. The magnitude of the output voltage is a property of the operational amplifier and its power supply.

Figure 28. Zero cross detector circuit

If the voltage remains higher than the reference voltage, 0V, the output is high. If the voltages becomes lower than 0V, the output drops down to the negative saturation value.
